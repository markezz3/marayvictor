# 💍 Mara & Víctor

Sitio web oficial de la boda de Mara y Víctor.

## 🌐 Sitio Web

**Producción:** https://marayvictor.com

## 📖 Acerca del proyecto

Este proyecto nació con el objetivo de crear un espacio digital donde familiares y amigos puedan encontrar toda la información relacionada con la boda de Mara y Víctor.

La idea es que el sitio funcione como:

* Invitación digital
* Centro de información para invitados
* Blog de preparativos
* Galería de fotografías
* Portal de confirmación de asistencia (RSVP)
* Recuerdo digital permanente después de la boda

## 📦 Visión a largo plazo

Una vez finalizada la boda, el sitio dejará de ser únicamente una invitación digital para convertirse en una cápsula del tiempo.

* Historia de la pareja

El objetivo es que el sitio permanezca como un recuerdo digital accesible para Mara y Víctor durante muchos años.
